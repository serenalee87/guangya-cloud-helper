光鸭云盘批量助手 V4 是一个油猴脚本，用于增强光鸭云盘网页端的批量整理、批量下载、磁力云添加、跨网盘秒传 JSON 生成与直接导入能力。

脚本面板里每个功能模块都带有简短步骤提示，新用户可以按模块顶部的小卡片操作。

## 功能总览

- 批量直链下载（光鸭）：读取光鸭当前勾选项，展开文件夹，逐个获取单文件直链并交给浏览器或下载器下载。
- 分享直读（夸克、123、天翼、迅雷）：粘贴分享链接，读取目录，勾选后生成光鸭秒传 JSON 或直接导入光鸭。
- 网盘互通（夸克、123、天翼、百度、迅雷）：从当前网盘页面抓取或主动读取文件信息，生成光鸭可导入的秒传 JSON。
- 磁力云批量添加（光鸭）：批量识别 TXT / JSON 里的 magnet 链接并提交光鸭云添加任务。
- 移动整理（光鸭）：读取当前网页已勾选项，整体上移、拆开文件夹内容，或移动到指定目录。
- 批量改名（光鸭）：预览并执行批量重命名，支持增加文字、替换文字、格式命名和自定义规则。
- 重复项清理（光鸭）：预览重复项，勾选确认后批量删除。
- 空目录扫描（光鸭）：递归扫描最里层空目录，并支持勾选删除。
- 高级与调试：手动认证、目录上下文、正则模板和调试信息兜底。

## 使用方式

### 1. 批量直链下载（光鸭）

用途：绕过光鸭网页端打包下载，逐个文件获取直链，让浏览器、IDM、NDM、Motrix 等本地下载器接管下载。

用法：

1. 在光鸭文件列表中打开目标文件夹。
2. 勾选要下载的文件或文件夹。
3. 打开脚本面板里的“批量直链下载（光鸭）”。
4. 点“读取当前勾选并展开”。
5. 确认待下载列表后，点“下载勾选”。

可用按钮：

- 读取当前勾选并展开：读取网页当前已勾选项，并递归展开文件夹。
- 下载勾选：获取直链后交给浏览器或下载器下载。
- 清空结果：清空当前待下载列表。
- 下载光鸭MD5/Size：导出当前勾选文件的光鸭 MD5 / Size 清单。

注意事项：

- 该功能只支持光鸭当前文件列表，不读取其他网盘分享链接。
- 建议先进入文件夹，再勾选里面的文件，识别更稳。
- 下载速度、多线程和断点续传由你的浏览器或下载器决定。
- 直链通常有时效，获取后建议尽快开始下载。
- 如果出现较多“源已修改”，可把“每批并发取链数”降到 `1` 或 `2` 后重试。

### 2. 分享直读（夸克、123、天翼、迅雷）

用途：不需要先保存到光鸭，直接读取其他网盘分享链接里的目录和文件，勾选后生成秒传 JSON 或直接导入光鸭。

用法：

1. 打开“分享直读（夸克、123、天翼、迅雷）”。
2. 粘贴分享链接；如有提取码 / 访问码，也可以一起粘贴或单独填写。
3. 点“读取链接目录”。
4. 在读取结果里全选、全不选或手动勾选文件。
5. 点“按勾选直接导入光鸭”，或点“按勾选生成 JSON”。

支持：

- 夸克分享链接
- 123 网盘分享链接
- 天翼云盘分享链接
- 迅雷云盘分享链接
- 自动识别复制文本里的链接和提取码
- 自动翻页或递归读取目录
- 按勾选结果直接导入光鸭

注意事项：

- 分享直读本质是用来源网盘返回的 MD5 / GCID + size 生成秒传信息，不上传文件内容。
- 夸克在部分情况下需要填写 Cookie 才能获取真实 MD5。
- 天翼分享如果需要访问码，请填写访问码。
- 迅雷分享直读需要先打开并加载对应迅雷分享页一次，让脚本捕获登录态和 `pass_code_token`。
- 直接导入光鸭需要光鸭 Authorization；可以先打开光鸭页面让脚本自动捕获，或手动粘贴 `Bearer ...`。

### 3. 网盘互通（夸克、123、天翼、百度、迅雷）

用途：在当前网盘网页中抓取或主动读取文件数据，生成光鸭可导入的秒传 JSON，并可直接导入光鸭。

用法：

1. 打开来源网盘页面。
2. 等待文件列表加载完成；如果是可勾选页面，先勾选目标文件。
3. 打开“网盘互通（夸克、123、天翼、百度、迅雷）”。
4. 点“从当前网页抓取生成”。
5. 生成成功后，可点“直接导入光鸭”“复制 JSON”或“下载 JSON”。

也可以：

1. 点“选择秒传 JSON”。
2. 选择已有 JSON 文件。
3. 点“生成可导入 JSON”。
4. 再复制、下载或直接导入光鸭。

支持说明：

- 夸克：支持分享页主动读取目录，并尽量获取真实 MD5。
- 123：支持当前分享页主动读取目录，也支持分享链接直读。
- 天翼：支持分享信息读取和访问码。
- 百度：百度分享页不能直接获取真实 MD5，建议先保存到自己的百度网盘，再在自己的百度网盘目录勾选生成。
- 迅雷：支持 GCID 路径；分享相关能力通常需要先打开迅雷分享页捕获登录态。

注意事项：

- 秒传是否成功取决于来源是否提供真实 MD5 / GCID，以及光鸭服务端是否命中库存。
- 秒传 JSON 不是文件内容，不等于上传文件。
- 光鸭未命中库存时，JSON 格式正确也可能导入失败。
- 直接导入需要光鸭 Authorization。
- 如果只想读一个分享链接，优先使用“分享直读”，流程更清晰。

### 4. 磁力云批量添加（光鸭）

用途：把本地 TXT / JSON 等文本文件里的 magnet 链接批量提交到光鸭云添加。

用法：

1. 打开“磁力云批量添加（光鸭）”。
2. 点“选择TXT/JSON”。
3. 选择包含 magnet 链接的文件。
4. 等脚本识别磁力数量。
5. 点“开始云添加”。
6. 可点“查看云任务”查看云添加任务列表。

支持：

- TXT / JSON / LOG / TEXT / MD 文本文件
- 自动识别 `magnet:` 链接
- 自动去重
- 自动创建导入目录
- 按每批最多文件数拆分提交
- 读取云任务历史，减少重复添加

注意事项：

- “云添加每批最多文件数”只影响拆批数量。
- 试用版或接口限制下，建议保持默认 `500`。
- “云添加目录前缀”只影响自动创建的目录名。

### 5. 移动整理（光鸭）

用途：批量整理光鸭文件列表里已经勾选的文件或文件夹。

用法：

1. 在光鸭网页文件列表中勾选文件或文件夹。
2. 打开“移动整理（光鸭）”。
3. 点“读取当前勾选”。
4. 根据需要选择：
   - 勾选项整体上移一层
   - 拆开文件夹内容到当前目录
   - 勾选项移到目标目录

功能说明：

- 勾选项整体上移一层：保留文件夹本身，把勾选项移到上一层目录。
- 拆开文件夹内容到当前目录：把文件夹里的直接内容提到当前目录，不保留外层文件夹。
- 勾选项移到目标目录：把勾选项移动到指定 `parentId` 目录。

注意事项：

- 这里读取的是网页文件列表里的真实勾选，不是面板里的勾选。
- “目标目录 parentId”需要填光鸭目标文件夹的目录 ID。
- 移动前建议先点“读取当前勾选”确认列表。

### 6. 批量改名（光鸭）

用途：批量修改光鸭文件或文件夹名称。

用法：

1. 打开“批量改名（光鸭）”。
2. 选择预处理规则，例如删除开头 `[] / 【】` 段、固定文字替换或不处理。
3. 选择最终改名方式，例如增加文字、替换文字、格式命名。
4. 在示例区域确认效果。
5. 点“预览”。
6. 确认无误后点“执行改名”。

支持：

- 删除开头第一个 `[]` / `【】` 段
- 固定文字查找替换
- 增加前缀或后缀
- 名称之后增加文字时可忽略后缀名，把文字加在扩展名前
- 最终查找替换
- 格式命名
- 自定义模板
- 自定义正则
- 保存配置

注意事项：

- 强烈建议先预览，再执行改名。
- 如果接口容易失败，可适当调大“每次请求间隔毫秒”。
- 自定义正则和自定义模板属于高级用法，不熟悉时建议先用普通选项。

### 7. 重复项清理（光鸭）

用途：识别并删除类似 `(1)`、`(2)`、`(3)` 结尾的重复项。

用法：

1. 打开“重复项清理（光鸭）”。
2. 根据需要调整“重复项编号”，默认 `1,2,3`。
3. 点“重复项预览”。
4. 点“勾选重复项”，或在列表里手动调整。
5. 确认后点“删除重复项”。

注意事项：

- 删除前请在列表里确认，不想删除的项目可以取消勾选。
- 该功能按名称规则识别重复项，不会判断文件内容是否完全相同。

### 8. 空目录扫描（光鸭）

用途：扫描当前目录树中最里层且完全为空的目录，并支持删除。

用法：

1. 打开“空目录扫描（光鸭）”。
2. 点“扫描空目录”。
3. 在结果列表里勾选或取消勾选。
4. 点“删除空目录”。

注意事项：

- 脚本只处理空目录，不会删除普通文件。
- 目录数量很大时扫描会需要更久。
- 删除前建议检查结果列表。

### 9. 高级与调试

用途：自动捕获失败时的兜底设置和排查工具。

包含：

- 刷新已捕获上下文
- 手动填写 Authorization
- 手动填写 DID / DT
- 手动填写 parentId
- 自定义模板
- 自定义正则
- 查看调试信息

注意事项：

- 普通使用通常不需要打开这里。
- 只有自动抓不到认证、目录或上下文时，再尝试手动填写。

## 常见注意事项

- 本脚本只增强网页端操作，不提供破解、绕过付费或上传文件内容的能力。
- 秒传依赖来源网盘提供的真实 MD5 / GCID + 文件大小，也依赖光鸭服务端库存命中。
- 直接导入光鸭前，需要光鸭 Authorization 可用。
- 浏览器下载是否能被 IDM、NDM、Motrix 等接管，取决于你本地下载器和浏览器扩展配置。
- 大目录、大批量任务建议分批操作，避免网页卡顿或接口限流。
- 直链下载失败时，可降低并发取链数后重试。
- 分享链接失效、访问码错误、平台接口变化都可能导致读取失败。
